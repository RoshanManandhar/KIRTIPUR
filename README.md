# Kirtipur_Tourism_Project

A simple, responsive tourism platform built with PHP & MySQL, allowing users to explore places, view photos, read details about attractions, and get in touch with the site admins. Features include an admin dashboard, user authentication, and place management.

## 🏯 Kirtipur Tourism Website

A simple **PHP-based tourism website** where users can explore **places and attractions** in Kirtipur, view photos, read detailed place information, and contact the site for more information. Admins can manage places, photos, and user activity.

## ✨ Features

- 🗺️ **Places & Attractions** – Browse tourist spots in Kirtipur, displayed in a card-based layout.
- 📍 **Place Details** – View in-depth information about each attraction.
- 🖼️ **Photo Gallery** – Upload and view photos of places.
- 📩 **Contact Form** – Users can submit inquiries or feedback directly.
- 👤 **User Dashboard** – Users can register, log in, and interact with the site.
- 🔑 **Admin Dashboard** – Admins can manage places, photos, and users.
- 🔒 **Authentication** – Separate login for users and admins.

## 🔧 Tech Stack

- **Backend:** PHP, MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Authentication:** PHP Sessions
